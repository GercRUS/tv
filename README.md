# Tiny IPTV Mono
Минималистичный плеер для Android-планшетов.

### Управление:
- **Переключение каналов:** Свайп вверх или вниз.
- **Список каналов:** Нажатие (тап) в левой трети экрана.
- **Настройки:** Кнопка-шестеренка в меню каналов.
- **Физические кнопки:** Поддержка кнопок "Вперед/Назад" на корпусе.
